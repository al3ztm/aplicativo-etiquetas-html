Lista Completa de Etiquetas HTML

Guía de referencia con ejemplos de código

Estructura Texto Formato Listas Enlaces Tablas Formularios Multimedia Semánticas Metadatos Programación

1. Estructura del Documento

<!DOCTYPE>

Define el tipo de documento HTML. Debe aparecer al inicio de cada documento HTML.

<!DOCTYPE html>

<html>

Elemento raíz de un documento HTML. Todos los demás elementos deben ser descendientes de este elemento.

<html lang="es">
    <!-- Contenido de la página -->
</html>

<head>

Contiene metadatos, enlaces a recursos externos y otra información sobre el documento.

<head>
    <meta charset="UTF-8">
    <title>Mi Sitio Web</title>
    <link rel="stylesheet" href="styles.css">
</head>

<body>

Contiene todo el contenido visible del documento HTML.

<body>
    <header>Encabezado</header>
    <main>Contenido principal</main>
    <footer>Pie de página</footer>
</body>

2. Metadatos

<meta>

Define metadatos sobre el documento HTML como la codificación, descripción, palabras clave, autor, etc.

<meta charset="UTF-8">
<meta name="description" content="Descripción de la página">
<meta name="viewport" content="width=device-width, initial-scale=1.0">

<title>

Define el título del documento que se muestra en la pestaña del navegador.

<title>Mi Página Web</title>

<link>

Establece relaciones entre el documento actual y recursos externos (normalmente CSS).

<link rel="stylesheet" href="styles.css">
<link rel="icon" href="favicon.ico" type="image/x-icon">

<style>

Contiene información de estilo CSS para el documento.

<style>
    body {
        font-family: Arial, sans-serif;
        margin: 0;
        padding: 20px;
    }
</style>

<base>

Especifica la URL base para todas las URLs relativas dentro del documento.

<base href="https://www.ejemplo.com/paginas/">

3. Texto y Encabezados

<h1> a <h6>

Encabezados de seis niveles diferentes, donde h1 es el más importante y h6 el menos importante.

<h1>Título Principal</h1>
<h2>Subtítulo</h2>
<h3>Sección</h3>
<h4>Subsección</h4>
<h5>Título menor</h5>
<h6>Título más pequeño</h6>

<p>

Define un párrafo de texto.

<p>Este es un párrafo de texto. Los párrafos son bloques de texto que el navegador separa automáticamente.</p>

<br>

Inserta un salto de línea simple (no crea un nuevo párrafo).

Esta línea<br>tiene un salto en medio.

<hr>

Crea una línea horizontal que representa un cambio temático en el contenido.

<p>Primer tema</p>
<hr>
<p>Segundo tema</p>

4. Formato de Texto

<strong>

Define texto con fuerte importancia, normalmente mostrado en negrita.

<p>Esta es una <strong>información importante</strong> que no debes olvidar.</p>

<em>

Define texto enfatizado, normalmente mostrado en cursiva.

<p>Debes prestar <em>especial atención</em> a este punto.</p>

<mark>

Define texto que debe ser marcado o resaltado por su relevancia.

<p>No olvides revisar <mark>esta sección</mark> antes del examen.</p>

<small>

Define texto más pequeño, como notas o comentarios secundarios.

<p>Precio: $99 <small>(impuestos no incluidos)</small></p>

<del>

Define texto que ha sido eliminado del documento, mostrado tachado.

<p><del>$50</del> $40 - ¡Oferta especial!</p>

<ins>

Define texto que ha sido insertado en el documento, mostrado subrayado.

<p>El evento será <del>el sábado</del> <ins>el domingo</ins>.</p>

<sub>

Define texto en subíndice, mostrado más pequeño y debajo de la línea de texto.

<p>La fórmula del agua es H<sub>2</sub>O.</p>

<sup>

Define texto en superíndice, mostrado más pequeño y encima de la línea de texto.

<p>3<sup>2</sup> = 9</p>

<code>

Define texto que representa código de programación.

<p>La función <code>console.log()</code> muestra mensajes en la consola.</p>

<pre>

Define texto preformateado que conserva espacios y saltos de línea.

<pre>
function saludar() {
    console.log("Hola mundo");
}
</pre>

<blockquote>

Define una sección que cita contenido de otra fuente.

<blockquote cite="https://www.ejemplo.com">
    <p>Esta es una cita extensa de otra fuente.</p>
</blockquote>

<q>

Define una cita corta en línea.

<p>Como dijo Einstein: <q>La imaginación es más importante que el conocimiento</q>.</p>

5. Listas

<ul>

Define una lista desordenada (con viñetas).

<ul>
    <li>Elemento 1</li>
    <li>Elemento 2</li>
    <li>Elemento 3</li>
</ul>

<ol>

Define una lista ordenada (numerada).

<ol>
    <li>Primer paso</li>
    <li>Segundo paso</li>
    <li>Tercer paso</li>
</ol>

<li>

Define un elemento de lista, usado dentro de <ul> o <ol>.

<li>Este es un elemento de lista</li>

<dl>

Define una lista de definiciones.

<dl>
    <dt>HTML</dt>
    <dd>HyperText Markup Language</dd>
    <dt>CSS</dt>
    <dd>Cascading Style Sheets</dd>
</dl>

<dt>

Define un término en una lista de definiciones.

<dt>HTML</dt>

<dd>

Define la descripción de un término en una lista de definiciones.

<dd>Lenguaje de marcado utilizado para crear páginas web.</dd>

6. Enlaces

<a>

Define un hipervínculo que enlaza a otra página o recurso.

<a href="https://www.ejemplo.com">Visita Ejemplo.com</a>
<a href="pagina2.html">Ir a Página 2</a>
<a href="mailto:correo@ejemplo.com">Enviar correo</a>
<a href="tel:+123456789">Llamar</a>

7. Multimedia

<img>

Inserta una imagen en el documento.

<img src="imagen.jpg" alt="Descripción de la imagen" width="300" height="200">

<figure>

Representa contenido independiente, a menudo con una imagen y una leyenda.

<figure>
    <img src="grafico.jpg" alt="Gráfico de ventas">
    <figcaption>Fig. 1: Ventas del primer trimestre 2025</figcaption>
</figure>

<figcaption>

Define una leyenda o título para una figura.

<figcaption>Fig. 1: Descripción de la imagen</figcaption>

<audio>

Inserta contenido de audio en el documento.

<audio controls>
    <source src="cancion.mp3" type="audio/mpeg">
    <source src="cancion.ogg" type="audio/ogg">
    Tu navegador no soporta el elemento de audio.
</audio>

<video>

Inserta contenido de video en el documento.

<video width="320" height="240" controls>
    <source src="video.mp4" type="video/mp4">
    <source src="video.webm" type="video/webm">
    Tu navegador no soporta el elemento de video.
</video>

<source>

Especifica recursos alternativos para elementos de audio o video.

<source src="video.mp4" type="video/mp4">

<track>

Especifica pistas de texto para elementos multimedia (como subtítulos).

<track src="subtitulos.vtt" kind="subtitles" srclang="es" label="Español">

<canvas>

Define un área de dibujo para gráficos con JavaScript.

<canvas id="miCanvas" width="200" height="100"></canvas>

<svg>

Define un contenedor para gráficos SVG (Scalable Vector Graphics).

<svg width="100" height="100">
    <circle cx="50" cy="50" r="40" stroke="black" stroke-width="2" fill="red" />
</svg>

8. Tablas

<table>

Define una tabla.

<table>
    <caption>Lista de empleados</caption>
    <thead>
        <tr>
            <th>Nombre</th>
            <th>Departamento</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Juan</td>
            <td>Marketing</td>
        </tr>
    </tbody>
</table>

<caption>

Define un título o leyenda para una tabla.

<caption>Tabla 1: Datos de ventas 2025</caption>

<thead>

Agrupa el contenido del encabezado de una tabla.

<thead>
    <tr>
        <th>Nombre</th>
        <th>Edad</th>
    </tr>
</thead>

<tbody>

Agrupa el contenido del cuerpo de una tabla.

<tbody>
    <tr>
        <td>María</td>
        <td>28</td>
    </tr>
</tbody>

<tfoot>

Agrupa el contenido del pie de una tabla.

<tfoot>
    <tr>
        <td colspan="2">Total: 10 empleados</td>
    </tr>
</tfoot>

<tr>

Define una fila de celdas en una tabla.

<tr>
    <td>Celda 1</td>
    <td>Celda 2</td>
</tr>

<th>

Define una celda de encabezado en una tabla.

<th>Nombre del producto</th>

<td>

Define una celda de datos en una tabla.

<td>Producto A</td>

<colgroup>

Especifica un grupo de una o más columnas para aplicar formato.

<colgroup>
    <col span="2" style="background-color: yellow">
    <col style="background-color: lightblue">
</colgroup>

<col>

Especifica propiedades para cada columna dentro de un <colgroup>.

<col style="width: 150px">

9. Formularios

<form>

Define un formulario HTML para entrada del usuario.

<form action="/procesar.php" method="post">
    <!-- Elementos del formulario -->
    <input type="submit" value="Enviar">
</form>

<input>

Define un campo de entrada donde el usuario puede ingresar datos.

<!-- Tipos de input -->
<input type="text" name="nombre" placeholder="Tu nombre">
<input type="email" name="email" placeholder="correo@ejemplo.com">
<input type="password" name="contraseña">
<input type="number" name="edad" min="18" max="99">
<input type="checkbox" name="acepto" value="si">
<input type="radio" name="genero" value="hombre">
<input type="date" name="fecha">
<input type="file" name="archivo">
<input type="submit" value="Enviar">
<input type="reset" value="Restablecer">

<textarea>

Define un control de entrada de texto multilínea.

<textarea name="comentario" rows="4" cols="50" placeholder="Escribe tu comentario aquí"></textarea>

<button>

Define un botón clickeable.

<button type="submit">Enviar</button>
<button type="reset">Restablecer</button>
<button type="button" onclick="miFuncion()">Haz clic</button>

<select>

Define una lista desplegable.

<select name="pais">
    <option value="">Selecciona un país</option>
    <option value="es">España</option>
    <option value="mx">México</option>
    <option value="ar">Argentina</option>
</select>

<option>

Define una opción en una lista desplegable.

<option value="es">España</option>

<optgroup>

Define un grupo de opciones relacionadas en una lista desplegable.

<select name="continente">
    <optgroup label="Europa">
        <option value="es">España</option>
        <option value="fr
